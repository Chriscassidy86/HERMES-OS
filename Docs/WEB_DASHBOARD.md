# Web Dashboard

Run `python scripts/read_only_dashboard.py DATABASE --port 8765`. The application rejects every bind host except `127.0.0.1`; expose it neither publicly nor through an unaudited proxy. Routes are GET-only, carry PAPER MODE, and provide presentation projections without order, secret, withdrawal, or configuration controls.

The dashboard reads application services. Frontend code does not fetch providers or calculate trading decisions.
