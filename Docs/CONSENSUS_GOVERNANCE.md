# Consensus Source Governance

Every source declares its category, homepage, approved access method,
authentication and licensing status, rate-limit note, coverage, update and
freshness expectations, reliability, implementation state, limitations, terms,
attribution, and historical-evaluation permission.

Sources fail closed unless explicitly activated. Deferred, unavailable, and
unconfigured licensed sources cannot activate. Fixtures remain labeled as
fixtures; imports require the operator to confirm access rights and preserve
attribution. Hermes stores normalized evidence and short compliant summaries,
not copyrighted articles or large copied posts.

Public popularity is not expertise, source registration is not endorsement, and
source performance never changes weights automatically. Consensus remains
advisory and cannot bypass the Risk Manager or cause PAPER execution.
