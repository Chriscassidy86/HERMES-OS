# Risk Philosophy

Risk Manager has final veto authority. Approval requires a validated directional
recommendation, finite bounded confidence, deterministic size limits, supported
paper execution, and complete evidence. Portfolio code independently enforces the
veto and cap. Uncertainty reduces exposure or produces no trade.

V2 is long-only simulated execution with fixed demonstration caps. Shorts, margin,
leverage, live accounts, withdrawals, and authenticated exchange actions are
prohibited. Risk changes are versioned research proposals requiring tests,
replay/paper evidence, human approval, and rollback instructions.
