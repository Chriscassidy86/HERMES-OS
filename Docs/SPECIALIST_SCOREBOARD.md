# Specialist Scoreboard

The scoreboard covers Trend, Market Regime, Momentum, Volatility, Volume, Liquidity,
Probability, and Portfolio Context specialists. Only included LONG/SHORT calls enter
directional accuracy. Neutral and excluded calls are reported separately. Symbol,
regime, timeframe, confidence, calibration, outcome grouping, streaks, and sample-size
warnings are deterministic. Outputs are recommendations only; no weight changes occur.
